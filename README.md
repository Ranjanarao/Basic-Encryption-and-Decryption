Basic Encryption & Decryption
A simple Python project that demonstrates text encryption and decryption using the Caesar Cipher algorithm.

Features
- Encrypt plain text using a shift key
- Decrypt encrypted text back to the original message
- Simple command-line interface
- Beginner-friendly implementation

Technologies Used
- Python 3
- Caesar Cipher

Usage
Run the program:
python encryption_decryption.py
Example:
Input: Hello World
Shift: 3

Encrypted Text:  Khoor Zruog
Decrypted Text:  Hello World

Learning Outcomes
- Basic Cryptography Concepts
- Encryption & Decryption Techniques
- Python Programming
- Cybersecurity Fundamentals
